- (String) run_command(*words)

Note:

Every argument is a shell word and will be escaped as such, unless a symbol.

Run a shell command.

Examples:

run_command('echo', 'foo bar baz', :>, file.path) will run ‘echo foo\ bar\ baz > '.

def run_command(*words)
  out = %x{#{words.map {|w| w.is_a?(Symbol) ? String(w) : String(w).shellescape }.join(' ')}}.chomp
  @exitstatus = $?.exitstatus
  out
end
